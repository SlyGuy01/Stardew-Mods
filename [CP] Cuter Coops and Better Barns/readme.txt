cuter coops and better barns

author: daisyniko
version: 1.0

http://www.nexusmods.com/stardewvalley/mods/4226



- installation -

1. install SMAPI and Content Patcher
2. download mod and open the .zip folder
3. extract mod to Stardew Valley/Mods
4. run the game using SMAPI



- config options -

"WallColour": Vanilla, Blue, Chocolate, Cream, Orange, PastelPink, Pink, Purple, Red, or Teal (default Blue)
